###############
Jobs Dashboard
###############

Introduction
------------

The **Jobs Dashboard** is a monitoring tool that displays all job executions for VedaOnline models.
It shows job history including user activities, execution modes, GitHub commit references, timing details, and 
completion status. Users can track various job types such as Solve, Sync, ReportCreate, ResultsCreate, DownloadViews, and 
TaskChain operations. Each job is linked to a specific GitHub commit for version control and traceability.

How to use it?
--------------

* Click **Jobs Dashboard** in the main menu.
* The Jobs Dashboard will display a list of jobs.
* Each job will display the job name, status, and progress.
* You can click on a job to view the details of the job.
